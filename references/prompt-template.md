# 虾线橘猫 Prompt 模板（即梦 v40 验证版）

> v40 API 验证通过的 prompt 格式。严格按此结构，不可自由发挥。

---

## 主生成 Prompt 结构

```text
简笔画风格，手绘线稿，[一只橘白相间的猫咪...在纯白背景中央]。

猫咪特征：
- 白色打底毛色，脸颊和额头有橘色斑块
- 背部正中央有一条从头到尾的纵向橘色条纹，这是天生的毛色
- 圆眼睛，[表情]，不卖萌，不笑
- 身体圆润但不过分肥胖
- 尾巴[尾巴姿态]

画面要求：
- 黑色细线勾勒轮廓，线条略带手绘抖动感
- 内部不上色，除了橘色条纹和橘色脸颊斑块
- 纯白背景，无纹理，无阴影，无渐变
- 大量留白，猫咪只占画面中间约40%
- [16比9横版 / 3比4竖版]构图
- 整体风格清爽、极简、有手绘草图感，不要精致插画感
```

---

## 生成命令

```bash
cd 凤梨/claude-code/skills/jimeng-batch && bun scripts/batch.ts "<manifest>" "<output>" 3:4 --ref "<参考图路径>"
```

- 模型：`jimeng_t2i_v40`
- 尺寸：小红书 `3:4`，贴图号 `16:9`
- `--ref` 传 `examples/images/` 中任一张 jimeng 原图作为风格锚定

---

## 关键约束

- **虾线必须在背上**：所有猫咪角度必须侧身或3/4，背部完整朝外。正面坐姿会让即梦把条纹画到胸前/腹部——绝对禁止。无例外。
- **内部不上色**：猫身体白色部分纯白，仅橘色条纹和脸颊斑块上色
- **无装饰元素**：不要气泡、手机、纸条、红虾、文字标签
- **手绘感**："线条略带手绘抖动感"必须保留
- 只用黑色和橘色，不用红蓝
