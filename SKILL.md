# 虾线橘猫 — IP 配图生成

> 基于 ian-xiaohei-illustrations 改编，将角色从小黑替换为虾线橘猫。
> 通过即梦4.0 API + 参考图锁定形象，确保每次生成风格一致。

## IP 角色

虾线橘猫 — 橘白相间的猫，背部正中央有一条纵向橘色条纹（虾线）。
不是吉祥物、不卖萌，是"正在认真参与系统运转的荒诞工作者"。

详细角色圣经见 `references/xiaohei-ip.md`。

## 使用方式

### 单张配图

```
帮我用虾线橘猫给这段话配一张图：信任不是喊出来的，而是一块证据一块证据铺过去。
```

Claude 会：分析内容 → 设计场景 → 调用即梦 API 生成 → 保存 PNG。

### 文章批量配图

```
用虾线橘猫给这篇文章配 5 张图，16:9 横版正文插图。
```

Claude 会：读文章 → 选 5 个认知锚点 → 逐个设计场景 → 批量生成。

## 技术实现

- **API**: 火山引擎即梦4.0 (`jimeng_t2i_v40`)
- **参考图**: 每次生成附带 01-front.png 作为风格参考
- **脚本**: `scripts/generate.ts` — 封装了 API 调用 + 参考图注入
- **环境**: 需要 `.env` 配置火山引擎密钥（同 jimeng-batch）

## 风格约束

- 3:4 竖版（贴图号标准）
- 纯白背景，无纹理无阴影无渐变
- 黑色细线手绘线稿，轻微抖动
- 内部不上色，仅橘色条纹 + 脸颊斑块
- 大量留白，主体占画面 40-50%
- 每张图只表达一个核心概念
- 虾线必须清晰可见
- 不写标题，不写结构类型标签
